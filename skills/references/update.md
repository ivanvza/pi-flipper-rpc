# update — Firmware Updater

Update, back up, and restore the device firmware and internal storage.

## Commands

| Command | Arguments | Description |
|---------|-----------|-------------|
| `update install` | `<path_to_update.fuf>` | Verifies and installs an update package. |
| `update backup` | `<path_to_backup.tar>` | Creates an internal storage backup. |
| `update restore` | `<path_to_backup.tar>` | Restores from an internal storage backup. |
