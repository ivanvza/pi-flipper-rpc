# info — Device Information

Displays detailed information about the device and power system.

## Commands

| Command | Description |
|---------|-------------|
| `info device` | Displays information about the device (hardware, firmware, etc.). |
| `info power` | Displays information about the power system. |
| `info power_debug` | Displays detailed information about the power system. |

## Notes

- `device_info` is a deprecated alias for `info device`.
- `!` is also an alias for `info device`.
